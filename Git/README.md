# Various Git tutorials

This is a collection of git and github formations, from beginner to advanced. It has the following documents:

- [EN] git_user_guide.pdf : A detailed overview of git and github, from the basic functions to the interface of the forge.
- [FR] formation_git_2024.pdf : An in-depth tutorial going into the more complicated aspects of git. While not exhaustive, it has a lot of intermediate knowledge, and reading the above overview is a prerequisite.
- [FR] formation_git_solutions.pdf : Walkthrough of the exercises suggested by the above document.
- [FR] diapo_formation_git.pdf : Companion slides for the two document above.
