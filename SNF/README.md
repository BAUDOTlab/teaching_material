# Similarity Network Fusion

The goal of this hands-on is to learn the SNF method and practice it on 4 different datasets. SNF is an analysis tool to work on multimodal datasets. Instead of comparing the data directly, it builts a similarity network between the samples of each modality before merging them into one network, representing the sample similarity across modalities.

The hands-on uses R and the SNF package, as well as Cytoscape for visualisation. The HTML file is the compiled version of the Rmd file, which contains all the instructions and code required to run this hands-on.
