# Team Teaching Repository

This repo stores the teaching materials of Anaïs Baudot's team. Each folder contains the different files and code for the topic. In addition, there is a zip file at the root of every folder containing the compressed version of every files.

## Table of Content

### [Similarity Network Fusion](/SNF)

A hands-on to master SNF techniques and visualisation through Cytoscape. The data are not included but available publicly online.

### [Git](/Git)

Multiple resources to learn how to use Git and Github or Gitlab, as PDF documents. Some include practical examples too.
